집합론

사용하는 기호

가운데쓰기 2항 술어기호 \(\in\)을 허용한다. 집합론에서 모든 것은 집합이다.

\(\forall x\)은 '모든 집합 x에 대하여'를 나타낸다.

\(\exists x\)은 '집합 x가 존재한다'를 나타낸다.

2항 술어기호 \(\subseteq\)을 다음과 같이 줄여쓰기로 허용한다 :

\(\forall t(t\in x \Rightarrow t \in a) \)
