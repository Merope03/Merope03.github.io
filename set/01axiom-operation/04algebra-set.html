집합 대수

합집합, 교집합, 차집합

1.10 분리 공리와 , 1.4 합집합 공리에 의하여 두 집합이 주어졌을 때 다음과 같은 유일한 집합들을 생성가능하다.

합집합 : \(A\cup B = \{x\ |\ x\in A \vee x\in B \}\)

교집합 : \(A\cap B = \{x\in A\ |\ x\in B \}\)

차집합 : \(A-B = \{x\in A\ |\ x\notin B \}\)

Note : \(A-B\)는 \(A\setminus B\)로 표기하기도 한다.

집합 대수

1.14~1.18은 \(\cup, \cap, \subseteq, -\)에 대한 기본적인 성질들이다. 이를 집합 대수(Algebra of sets)라고 한다. 사실은 \(\forall A,B,C\)를 써야 하지만, 이는 생략하였다.

1.14. 교환 법칙

\(A\cup B = B\cup A\)
\(A\cap B = B\cap A\)

설명.

A에 B를 합집합하든, B에 A를 합집합하든 결과가 같다. 교집합도 마찬가지이다.

증명.

\(x\in A\cup B\)를 가정하자. 그러면 \(x\in A\) 또는 \(x\in B\)를 얻는다. 이는 \(x\in B\cup A\)를 함의하며, 더 자세한 설명은 생략한다. 1.1 외연 공리를 사용함에 주목하자.

1.15. 결합 법칙

\(A\cup (B\cup C) = (A\cup B)\cup C\)
\(A\cap (B\cap C) = (A\cap B)\cap C\)

설명.

A에 B를 합집합하고 그 후에 C를 합집합하든, B와 C를 합집합하고 A를 합집합하든 결과는 변하지 않는다.

증명.

\(x\in A\cup (B\cup C)\)를 가정하자.
그러면 \(x\in A\) 또는 \(x\in B\cup C\)이다.
\(x\in A\)인 경우, \(x\in A\cup B\)이고, 따라서 \( x\in (A\cup B)\cup C \)이다.
\(x\in B\cup C\)인 경우, \(x\in B\) 또는 \(x\in C\)이다.
\(x\in B\)인 경우, \(x\in A\cup B\)이고, 따라서 \( x\in (A\cup B)\cup C \)이다.
\(x\in C\)인 경우, \( x\in (A\cup B)\cup C \)이다.
따라서 그 어떤 경우에도 \( x\in (A\cup B)\cup C \)이다.
같은 방식으로 하면, 외연 공리에 의해 결국 원하는 결론을 얻는다.

1.16. 분배 법칙

\(A\cap (B\cup C) = (A\cap B)\cup(A\cap C)\)

\(A\cup (B\cap C) = (A\cup B)\cap(A\cup C)\)

설명.

덧셈과 곱셈의 분배법칙처럼, 교집합과 합집합의 분배법칙이 성립한다.

증명.

\(x\in A\cap (B\cup C)\)을 가정하자.
\(x\in A\wedge x\in B\cup C\)를 얻는다.
\(x\in B\cup C\)로부터, \(x\in B\vee x\in C\)이다. \(x\in B\)인 경우, \(x\in A \wedge x\in B\)이므로 \(x\in (A\cap B)\cup(A\cap C)\)이다.
\(x\in C\)인 경우도 마찬가지이다.
역으로 \(x\in (A\cap B)\cup(A\cap C)\)을 가정하자.
\(x\in A\cap B\)인 경우, \(x\in A\)이고 \(x\in B\)이므로 \(x\in A\cap (B\cup C)\)이다.
\(x\in A\cap C\)인 경우도 마찬가지이다.

1.17. 드 모르간

\(C-(A\cup B) = (C-A)\cap (C-B)\)
\(C-(A\cap B) = (C-A)\cup (C-B)\)

설명.

전체 집합을 C라고 두었을 때, 우리가 알고 있는 여집합의 개념을 사용하면 합집합의 여집합은 여집합의 교집합, 교집합의 여집합은 여집합의 합집합이 된다.

증명.

\(x\in C-(A\cup B)\)를 가정하자.
\(x\in C \wedge x \notin (A\cup B)\)를 얻는다.
\(x\in A\)인 경우, \(x\in A\cup B\)이므로 모순이다. 따라서 \(x\notin A\)이다.
같은 방식으로, \(x\notin B\)이다.
이로부터 \(x\in (C-A)\cap (C-B)\)를 얻는다.
역으로 \(x\in (C-A)\cap (C-B)\)를 가정하자.
\(x\in (C-A)\wedge x\in (C-B)\)이다.
이로부터 \(x\in C\)는 당연하다.
\(x\notin A\)이고 \(x\notin B\)도 얻을 수 있다.
만약 \(x\in A\cup B\)이면 이는 윗줄과 모순이므로, \(x\notin A\cup B\)이다.
따라서 \(x\in C-(A\cup B)\)이다.

1.18. \(\emptyset\)에 관한 항등식

\(A\cup \emptyset = A\)
\(A\cap \emptyset = \emptyset\)
\(A\cap(C-A) = \emptyset\)

증명.

공집합과 관련된 성질들은, 대개 공허참으로 증명할 수 있다.
첫 번째 성질만 증명하겠다.
\(x\in A\cup \emptyset\)을 가정하자.
\(x\in A\)인 경우, 더 이상 보일 것이 없다.
\(x\in \emptyset\)인 경우, 애초에 모순이다. 따라서 \(x\in A\)이다.
역방향은 당연하다.

여집합

집합 S가 주어져서, 이를 전체집합이라고 하자. 곧, 우리가 다루는 모든 집합은 S의 부분집합이라고 하자.
이 경우, 다음과 같이 여집합을 정의한다.
\(A^c = S-A\)

1.19. 전체집합과 관련된 성질들

A가 S의 부분집합이라 하자.
\(A\cup S = S\)
\(A\cap S = A\)
\(A\cup A^c = S\)
\(A\cap A^c = \emptyset\)

증명.

(1) \(x\in A\cup S\)를 가정하자.
\(x\in A\)인 경우, 부분집합의 정의에 의하여, \(x\in S\)이다.
\(x\in S\)인 경우, 더이상 보일 것이 없다. 역방향은 간단하다.
(2) \(x\in A\cap S\)를 가정하자.
\(x\in A\)로부터, 한 방향이 완료되었다.
\(x\in A\)를 가정하자. \(A\subseteq S\)이므로, \(x\in S\)이다. 따라서 증명이 완료되었다.
(3) \(x\in A \cup A^c\)를 가정하자. 둘 모두 S의 부분집합이므로, \(x\in S\)이다.
역으로 \(x\in S\)를 가정하자. 배중률에 의하여 \(x\in A\)또는 \(x\notin A\)인데, 각각의 경우에 증명이 완료된다.
(4) \(x \in A\)이고 \(x\notin A\)일 수는 없으므로, 공집합이다.
